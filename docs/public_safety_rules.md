# Public Safety Rules

This website is public-facing and must remain sanitized.

## Do Not Include

- Proprietary code
- Internal URLs
- Internal system names
- Real production screenshots
- Real data
- Actual chemical names
- Exact regulatory thresholds
- Exact financial values
- Customer names
- Vendor names
- External audit names
- Employee/user names
- Exact warehouse/bin/location names
- Exact item IDs
- Protected operational history

## Allowed Evidence Types

- Sanitized architecture summaries
- Synthetic schemas
- Synthetic examples
- Pseudocode
- Public-safe ERDs
- Generalized workflow descriptions
- Course-to-work synthesis
- Reflection and professional narrative
